КРАТКОЕ ОПИСАНИЕ:
этот проект представляет собой лендинг-страницу для школы иностранного языка. 

ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

*HTML: основной язык разметки для создания структуры страницы.
*CSS: каскадные таблицы стилей используются для стилизации и внешнего оформления страницы.
*Font Awesome: иконки применяются для улучшения визуального представления элементов на странице.
*Google Fonts: Шрифт Nunito Sans загружается из Google Fonts.

ОСОБЕННОСТИ ПРОЕКТА:

*Страница имеет ширину контента 1024px и выравнивается по центру страницы, обеспечивая приятное визуальное восприятие.
*Верстка стремится к максимальному соответствию макету, с учетом плавных отступов и выравнивания.
*Рабочее меню вверху страницы с якорными ссылками обеспечивает навигацию по разделам сайта.
*Кнопки на странице изменяют цвет при наведении и нажатии, что повышает интерактивность.
*Ссылки в меню и в футере также изменяют цвет при наведении для более привлекательного внешнего вида.

### ИНСТРУКЦИИ ПО УСТАНОВКЕ:

1. Склонируйте репозиторий на свой компьютер.
2. Откройте `index.html` в вашем веб-браузере.
